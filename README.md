# 🐛 Insect Prediction using Computer Vision

## 🚀 Overview

This project is a Computer Vision-based insect classification system that predicts insect species from images using deep learning. It uses a MobileNet-based architecture for efficient and accurate image classification.

---

## 🎯 Features

* 🖼️ Image-based insect classification
* 🤖 Deep learning model (MobileNet)
* ⚡ Fast prediction pipeline
* 📊 Dataset preprocessing and training support

---

## 🛠️ Tech Stack

* Python
* PyTorch
* OpenCV
* NumPy

## ▶️ How to Run

### 1. Install dependencies


### 2. Run prediction

```
python predict_insect.py
```
